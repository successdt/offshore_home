Enfinity is an elegant and flexible WordPress e-commerce themewith an adaptive layout and many features inside.Did you try its (really)powerful admin panel? Click here



Enfinity is the perfect theme if you need an ecommerce, a portfolio, a business site, a BBPress forum or a simple blog

See the products page

Navigate these slides to see the potential of one of the many features of this theme, the new slideshow generator
Enfinity supports (only to name the most famous ones) Camera slideshow, WooCommerce, BBPress, NextGen Gallery, WP Super Cache and WPMU
Pause Play Prev Next

Extended admin panel

To customize almost everything

Enter the admin panel and set your options.N.B.: if you are logged, the demo version could be slower than the original one, because the options saved are stored into the session and printed directly on the page, to test the real speed please log out.

E-commerce ready

By using Woo-Commerce

Enfinity supports WooCommerce plugin, but also (only to name the most famous ones) Cameraslideshow (developed by me and highly recommended), BBPress to use it as a forum, Wp Super Cache, NextGEN Gallery…Also tested with WPMU.

Perfect as a portfolio

With a like button integrated

Enfinity could be used as a portfolio, with a grid layout and the simple like button integrated, based on the plugin Like This. Portfolio pages supports masonry effect, ColorBox, videos allowed in the ColorBox, external links allowed too…

So many features inside

Shortcodes, widgets, megamenu…

Enfinity provides a lot of shortcodesalso available as TinyMCE buttons, resident widgets, a mega menu, very useful with an e-commerce,contact form andprice table generators, custom fullscreen backgrounds…
