Retro – Premium Vintage WordPress Theme

Live Preview Screenshots

Retro Premium WordPress Theme is a combination of a strong vintage design and powerful admin panel. Retro offers full control over site color scheme as well as 5 predefined skins, the fancy Revolution Slider, custom portfolio & testimonials post types, various shortcodes, custom widgets…

We have developed 5 awesome predefined skins to help you ease site creation process.
Any of them can be used as a start point and changed up to your needs.

Feeling not that confident with CSS? With Retro theme you can manage site color scheme right from the Admin Panel. Extensive editing options that come with the theme will help you adjust every piece of your site fast & easy.






Slider Revolution is a fully developed slide displaying system offering the capability to show images, videos and captions paired with simple, modern and fancy 3D transitions.

You can easily build your slides moving slide elements around with a mouse.
Besides you can check how a preview looks in real time.







Now you can change menu appearance just moving logo around!
Along with it you can easily change menu color options.





You do not need to install any third-party plugins to create a form – we’ve added a contact form builder right to the default WordPress content editor both for posts & pages.

To add a contact form to a page just click on its shortcode button in the Visual mode
and start customizing.




Our theme comes with two different layouts for blog listing (wide images) and shortcode (small images). Also you can choose to use one of these layout for categories.




Our theme is translation-ready. You can translate it to any language as well as set it up for multiple languages. It includes a .POT file that is used to create .po/.mo files in the language you need.

Make it multilingual with WPML

WPML makes it easy to run a multilingual website with a single WordPress install.
Choose languages you’d like to use for your site and start translating content.
You can arrange different language contents in the same domain (in language directories),
in sub-domains or in completely different domains.




We have added extensive font options to help you adjust theme look up to your needs.
Now you can pick between dozens of fonts to apply to different parts of the theme.
Powered by the free Google Fonts library, you have access to the best web fonts around that can be easily applied to a site at the click of a button.

can comfortably take charge of site creation process.

All of the settings come with hint messages and are logically arranged in groups.

So you always know what you have changed & where to find a needed option.
We hope such an approach will turn your site creation process into a pleasant experience.
